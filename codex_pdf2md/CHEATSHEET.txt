------------------------------------------------------------
SCRIPTS
------------------------------------------------------------

  pdf_to_md.py:
    python scripts/pdf_to_md.py --input data/current/<file>.pdf
    python scripts/pdf_to_md.py --all
    python scripts/pdf_to_md.py --input data/current/<file>.pdf --overwrite
    python scripts/pdf_to_md.py --input data/current/<file>.pdf --ocr-mode auto

  Key behavior:
    - Default mode is single-file only via --input
    - Batch mode requires --all
    - Existing generated book outputs are protected unless --overwrite is passed
    - Stable book IDs come from data/metadata/books_manifest.csv
    - Primary extraction uses PyMuPDF
    - OCR prefers OCRmyPDF when Tesseract is available
    - OCRmyPDF runs with the pypdfium rasterizer and optimization disabled, so Ghostscript is optional
    - Fallback OCR uses PyMuPDF integrated OCR when Tesseract is available
    - Arabic-heavy pages are re-OCRed directly with Tesseract to recover logical-order text instead of relying on the PDF text layer
    - Passage metadata now records coarse OCR confidence and weak-section review notes

  Generated outputs:
    - output/raw_text/<book_id>.txt
    - output/book_markdown/<book_id>.md
    - output/passage_markdown/<book_id>/<passage>.md
    - output/metadata/books.csv
    - output/metadata/passages.csv
    - output/review_logs/<book_id>.md
    - output/qc/<book_id>.json
    - output/qc/<book_id>_pages.csv
    - output/review_images/<book_id>/page_<nnn>.png
